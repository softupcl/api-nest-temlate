# Tienda API
1. Clonar proyecto
2. ```yarn install```
3. Clonar el archivos ```.env.template``` y renombrarlo a ```.env```
4. Cambiar las variables de entonor
5. Levantar base de datos en Docker
```
docker-compose up -d
```
6. Levantar modo desarrollos: ```yarn start:dev```
